# Oravox

A modern premium audio e-commerce website built with React.

## Features

- Responsive Design
- Shopping Cart
- Product Pages
- Newsletter Section
- Modern UI
- Local Storage Cart

## Installation

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build

```bash
npm run build
```

Preview

```bash
npm run preview
```

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
